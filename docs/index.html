  • CIVA - Carteira Internacional de Vacinação

    Projeto desenvolvido ao longo do curso de desenvolvedor Full Stack promovido pela FORSOFT Academy. Nele atuamos como uma empresa para elaborar uma plataforma que funciona como uma carteira de vacinações, onde dependendo do seu nível de acesso (portador ou um dos perfis de gestão) é possível consultar, validar, cadastrar e vincular / desvincular vacinas, unidades e pessoas. Tecnologias utilizadas: HTML5, CSS3, Bootstrap, Javascript, jQuery, AJAX, JAVA e SQL

    Link para a página do projeto ios_share
  • question_answer letmeask

    Projeto desenvolvido juntamente com a Rocketseat no evento NLW Together, e funciona como uma plataforma de perguntas e respostas, onde um usuário pode criar uma sala utilizando seu e-mail do google, escolher um nome para essa sala e compartilhar um código que é gerado automaticamente com outras pessoas para que elas possam entrar nessa sala e enviar perguntas. Tecnologias utilizadas: ReactJS, Typescript, SASS, Yarn e Firebase

    Link para o repositório no GitHub ios_share Link para a página do projeto ios_share
  • calculate Calculadora de Taxas

    Projeto desenvolvido de acordo com desafio da plataforma Frontend Mentor, que consiste em calcular o valor de taxas de serviço (ou outro tipos) sobre um valor informado em um dos campos da aplicação. É possível também dividir os valores de forma igual para um número de pessoas informado pelo usuário. Foram aplicados: HTML5, SASS, javascript vanilla e design responsivo.

    Link para o repositório no GitHub ios_share
  • Kenzietter

    Projeto desenvolvido em workshop da Kenzie Academy, reproduzindo a página de feed do Twitter. Nele, é possível escrever novos tweets, que quando são enviados, são adicionados à lista que fica logo abaixo, mostrando também a hora da postagem e os dados de perfil do usuário (nickname e foto do perfil). Foram utilizados: HTML5, CSS3 e Javascript vanilla

    Link para o repositório no GitHub ios_share
  • movie AluraFlix

    Projeto desenvolvido na Imersão Dev - Alura 2021, composto por uma página onde você deve colar no campo de texto o endereço da imagem de um filme que você goste, e depois clicar no botão "Adicionar Filme". Cada imagem será adicionada ao catálogo. Foram usados: HTML5, SCSS e Javascript vanilla
